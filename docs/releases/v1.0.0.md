# Claude Code Switcher 1.0.0

First public release.

## Highlights

- Built-in backend presets:
  - Claude subscription
  - DeepSeek V4 Pro
  - DeepSeek V4 Flash
- Custom Anthropic-compatible backend profiles.
- Per-backend API keys stored in macOS Keychain.
- Claude Code version check and update buttons.
- Claude Code Skill manager with category filtering, summaries, usage guidance, pause/resume, reveal, uninstall, and update checks.
- Optional Skill summary generation using a selected configured backend.

## Requirements

- macOS 14 or later.
- Claude Code installed and available as `claude`.

## Install

1. Download `Claude-Code-Switcher-macOS.zip`.
2. Unzip it.
3. Drag `Claude Code Switcher.app` into `/Applications`.
4. If macOS blocks the first launch, right-click the app and choose Open.

## Notes

- This build is ad-hoc signed for public source distribution, not notarized.
- Custom backend support is Anthropic-compatible only. It does not convert OpenAI, Gemini, or Ollama protocols.
- The app is macOS-only.

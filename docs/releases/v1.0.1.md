# Claude Code Switcher 1.0.1

Small language and packaging update.

## Highlights

- Added an in-app Chinese/English language switcher. Chinese remains the default.
- Window title, main navigation, backend controls, and Skill management controls follow the selected language.
- Skill usage guidance switches between Chinese and English.
- Generated Skill summaries now request the selected UI language and keep separate summary caches per language.
- Release packaging now includes localization resources in the `.app` bundle.

## Install

1. Download `Claude-Code-Switcher-macOS.zip`.
2. Unzip it.
3. Drag `Claude Code Switcher.app` into `/Applications`.
4. If macOS blocks the first launch, right-click the app and choose Open.

## Notes

- This build is ad-hoc signed for public source distribution, not notarized.
- The app is macOS-only.

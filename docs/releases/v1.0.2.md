# Claude Code Switcher 1.0.2

Bug-fix release for localization and Skill summaries.

## Highlights

- Fixed Chinese/English switching across backend cards, Skill controls, status lines, and window titles.
- Reworked localization so dynamic status messages are rendered from the current UI language instead of storing already-translated strings.
- Summary model selection no longer regenerates summaries automatically.
- Skill summaries are now cache-first and explicit: use Generate/Regenerate Summary when you want to call the selected backend.
- Newly discovered Skills can still be summarized automatically when a summary backend is configured.
- Summary failures now include clearer hints for missing keys, HTTP status codes, endpoint/model mismatch, and rate limits.

## Install

1. Download `Claude-Code-Switcher-macOS.zip`.
2. Unzip it.
3. Drag `Claude Code Switcher.app` into `/Applications`.
4. If macOS blocks the first launch, right-click the app and choose Open.

## Notes

- This build is ad-hoc signed for public source distribution, not notarized.
- The app is macOS-only.

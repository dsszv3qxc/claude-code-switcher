# Claude Code Switcher 1.0.3

Bug-fix release for Claude Code detection, installation, and Skill summaries.

## Highlights

- Fixed version checks from the macOS app when `claude` is installed in common user paths such as `~/.local/bin`.
- Version lookup now uses Claude's official latest-release endpoint first, with npm lookup only as a fallback.
- Update now runs `claude update` in the background instead of assuming an npm install.
- If Claude Code CLI is not found, the app now shows an install button that runs Claude's official native installer.
- Skill summaries now use a larger response budget, retry once if the model output is truncated, and support more Anthropic-compatible response shapes.
- Old truncated summary cache entries are ignored by a new summary cache version.
- Skill details now include a full Summary section so generated text and error details are not hidden.

## Install

1. Download `Claude-Code-Switcher-macOS.zip`.
2. Unzip it.
3. Drag `Claude Code Switcher.app` into `/Applications`.
4. If macOS blocks the first launch, right-click the app and choose Open.

## Notes

- This build is ad-hoc signed for public source distribution, not notarized.
- The app is macOS-only.

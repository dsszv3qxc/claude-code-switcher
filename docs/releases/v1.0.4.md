# Claude Code Switcher 1.0.4

Small feature release for Skill summaries.

## Highlights

- Added `批量生成摘要` / `Generate Summaries` to the Skill library menu.
- Batch generation uses the current category filter, so `全部分类` generates for all visible Skills and a selected category generates only that group.
- Existing generated summaries are skipped to avoid unnecessary API calls.

## Install

1. Download `Claude-Code-Switcher-macOS.zip`.
2. Unzip it.
3. Drag `Claude Code Switcher.app` into `/Applications`.
4. If macOS blocks the first launch, right-click the app and choose Open.

## Notes

- This build is ad-hoc signed for public source distribution, not notarized.
- The app is macOS-only.

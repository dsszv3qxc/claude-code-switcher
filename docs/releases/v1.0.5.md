# Claude Code Switcher 1.0.5

Feature release for Claude Code Effort defaults.

## Highlights

- Added a global Effort control to the backend status panel.
- Supported levels: Auto, Low, Medium, High, XHigh, and Max.
- Low through XHigh are saved to Claude Code's persistent `effortLevel` setting.
- Max is saved through `CLAUDE_CODE_EFFORT_LEVEL=max`, matching Claude Code's environment override path.
- Auto removes the app's global Effort override so Claude Code can follow its model/default behavior.

## Install

1. Download `Claude-Code-Switcher-macOS.zip`.
2. Unzip it.
3. Drag `Claude Code Switcher.app` into `/Applications`.
4. If macOS blocks the first launch, right-click the app and choose Open.

## Notes

- The setting affects newly started Claude Code sessions.
- This build is ad-hoc signed for public source distribution, not notarized.
- The app is macOS-only.

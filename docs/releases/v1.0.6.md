# Claude Code Switcher 1.0.6

Feature release for local Ollama Qwen backends.

## Highlights

- Dynamically discovers local Qwen models from Ollama's `http://localhost:11434/api/tags` endpoint.
- Adds discovered Qwen models to the backend picker as local Ollama options.
- Uses Ollama's official Anthropic-compatible API path for Claude Code.
- Lets local Ollama Qwen models generate Skill summaries from the Summary Model menu.
- Uses the local `ollama` auth token automatically, so no API key field is shown for Ollama models.
- Keeps the backend page usable when many local models are available by making the backend area scrollable.

## Install

1. Download `Claude-Code-Switcher-macOS.zip`.
2. Unzip it.
3. Drag `Claude Code Switcher.app` into `/Applications`.
4. If macOS blocks the first launch, right-click the app and choose Open.

## Notes

- Ollama must be running for local Qwen models to appear.
- The setting affects newly started Claude Code sessions.
- This build is ad-hoc signed for public source distribution, not notarized.
- The app is macOS-only.
